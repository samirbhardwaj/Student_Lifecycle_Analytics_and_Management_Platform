# Student Lifecycle Analytics & Management Platform

A complete advanced Java Web Application built using JSP, Servlet, JDBC and MySQL.

---

# 📌 Project Overview

The Student Lifecycle Analytics & Management Platform is an advanced web-based application developed to manage student data, internships, certifications, and analytics efficiently.

This project follows MVC Architecture using JSP, Servlet, JDBC, and MySQL.
